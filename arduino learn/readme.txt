Основное руководство содержится в файле Руководство.pdf. Для его чтения необходима
программа Adobe Reader. Вы можете скачать ее из интернета, либо запустить
установщик Software\AdbeRdr11000_ru_RU.exe.

Описание папок:
- Sketches: все используемые в руководстве исходные коды программ
- Libraries: используемые библиотеки Arduino
- Datasheets: техническая документация к некоторым компонентам набора
